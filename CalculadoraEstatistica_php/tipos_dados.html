2 - Dados Estátisticos

Um dos objetivos da Estatistica Descritiva é obter uma relação na quantidade de dados com os quais devemos operar. Esses dados são contruidos de duas formas: Variáveis e Tipos de Dados

1 - Variáveis
As variávies presentes nessa calculadora são todas variáveis quantitativas, ou seja, usamos a representação numérica. Elas podem ser classificadas em discretas e contínuas.

  • Continua: Qualquer valor entre dois limites. Se costuma fazer divisões, são originadas das medições. Os valores dessavariável podem assumir qualquer valor (inteiros e     fracionários) dentro do intervalo. A abrangência pode se caracterizar por infinitos valores.
  • Discreta: variável que assume valores geralmente inteiros, estando em dados brutos ou rol. Acontecem relacionadas a situações limitadas, não estando dentro de um     intervalo.

    2 - Tipos de Dados
    Antes de fazer qualquer calculo, é preciso obter os dados que serão processados. Essa calculadora trabalha com dois tipos da dados: Dados Brutos e Dados em Rol.
  • Dados Brutos: São os primeiros dados obtidos. Esses dados são inseridos de maneira aleatoria e desordenada;
  • Dados em Rol: É a organização dos dados brutos por ordem de valor, sendo ele crescente ou decrescente. É mais comum organizar em ordem crescente.
